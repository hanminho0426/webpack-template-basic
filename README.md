## [WEBPACK_TEMPLATE_BASIC](/ "Netlify로 이동")

- `이스케이프 문자(Escape String)`는 백슬래시를 사용해 원래의 의미를 벗어나는 문자를 만들 수 있다.   
정규표현식에서 .(마침표)은 '임의의 한 문자'를 의미하기 때문에, 그 의미에서 벗어나 단순 마침표로 해석될 수 있도록 이스케이프 문자로 만들어줘야 한다.
